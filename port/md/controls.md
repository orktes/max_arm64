
## Controls

| Button            | Description      |
|-------------------|------------------|
| A                 | Jump             |
| B                 | Crouch           |
| X                 | Reload           |
| Y                 | Use/Zoom         |
| L1                | Bullet time      |
| R1                | Shoot            |
| L2                | Previous Weapon  |
| R2                | Next Weapon      |
| D-Up              | Pain Killer      |
| D-Down            | Reload           |
| D-Left            | Previous Weapon  |
| D-Right           | Next Weapon      |
| Select+R1         | Quicksave        |
| Start             | Menu             |
| Left Stick        | Move             |
| Right Stick.      | Look/Aim         |
| Select+UP/DOWN    | Adjust Camera Y  |
| Select+LEFT/RIGHT | Adjust Camera X  |
